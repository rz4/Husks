Dependency Rules:
─────────────────
✓ Imports flow DOWNWARD only (higher → lower layers)
✓ Same-layer imports follow declared intra-layer ordering
✓ No cycles (Tarjan SCC = 0)
✓ TYPE_CHECKING for type-only imports that would violate layers

Special Modules:
────────────────
• utils.events, utils.console: Pure infra (no husks.* imports)
• gate: Isolated L0 (zero cross-dependencies by design)

Remaining Deferred Imports (9):
────────────────────────────────
All are performance optimizations to defer expensive subsystems:
• locke → core, designs.transport (parser heavy deps)
• designs.ir → build.*, locke (compiler heavy deps)  
• setup → cli.surface, resources, utils.console (init heavy deps)
• cli.cmd.validate → setup (internal helper)
